## Verification Rules

- Use current docs, not cached snippets, for exact commands and schemas.
- Check the changelog when a command, platform, or service seems different from an example.
- Query the caller's project for IDs, quota, and capacity.
- Query Token Factory's models endpoint for model IDs and capabilities.
- Check pricing immediately before quoting cost.
- Treat preview services and fields as change-prone and label them clearly.
